# PLAY-TINO-BE
Play-Tino Website Backend Project by using `Spring Boot`

# Skills
- Spring Boot

# Installation

## Using docker
### Go to play-tino directory
```bash
cd ./play-tino
```
### Build docker image
```bash
docker compose build
```
### Run in background
```bash
docker compose up
```

### Show logs
```bash
docker compose logs -f
```
